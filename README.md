# TransitIQ 🚍

AI-Powered Urban Transit Intelligence Platform

## Overview

TransitIQ predicts public transport delays using weather conditions, traffic congestion, city events, and temporal patterns.

The platform provides:

- Delay duration prediction
- Delay risk scoring
- Explainable AI insights
- Transit reliability analytics
- Smart departure recommendations

## Problem Statement

Public transport delays are influenced by multiple external factors including weather, traffic, holidays, and large-scale city events.

The goal of this project is to build an end-to-end machine learning system capable of forecasting delays and identifying the most influential contributing factors.
